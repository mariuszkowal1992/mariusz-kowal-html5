# MARIUSZ KOWAL - HTML5 Portfolio

A static portfolio website template built with HTML5, CSS3, and JavaScript. This project showcases the structure of a personal website tailored for a frontend developer, WordPress developer, and freelancer.

This repository was uploaded to GitHub to demonstrate code quality, file organization, and responsive layout construction. The template is not intended for download, redistribution, or commercial use.

## Project Purpose

The goal of this project is to present sample code for a static portfolio template. It serves as a foundation for future WordPress theme development or a personal presentation site.

The project includes standard portfolio views:

- Homepage
- Blog overview
- Search results
- Single blog post
- Portfolio grid
- Single project showcase
- Universal content page

## Technologies

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## File Structure

```text

|-- index.html
|-- blog.html

|-- page.html
`-- assets
    |-- css
    |   `-- styles.css
    |-- js
    |   `-- main.js
    `-- img
        `-- mariusz-kowal.webp
```

## Key Features

- Responsive mobile navigation
- Fullscreen hamburger menu
- Active navigation links based on scroll section
- Two-column hero section layout
- Cards for services, projects, and posts
- Pagination for blog and portfolio pages
- Category filtering for portfolio projects
- Search results page utilizing the `s` parameter
- Project screenshot carousel with thumbnails
- Working contact form layout
- WordPress-style comment layout
- Sidebar with integrated widgets
- Codebase prepped for seamless WordPress integration

## Usage Disclaimer

This code is hosted in this repository strictly for presentation and portfolio purposes. The template is not available as a free or commercial product for download or public redistribution.
